# API Documentation Reference

This directory contains local copies of Cribl API documentation to assist with development.

*   `cribl_api_intro.md`: Introduction to the Cribl API.
*   `cribl_api_authentication.md`: Details on authentication methods.
*   `cribl_api_update_configs.md`: Instructions for updating configurations.
*   `cribl-apidocs-*.yml`: The OpenAPI/Swagger definition file.

These documents are for reference purposes when working on the `cribl_api.py` client.
